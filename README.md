Aplikasi TugasMobile_1 (Hello UI) adalah proyek android sederhana yang dibuat untuk memenuhi tugas matakuliah Pemrograman Mobile. Fitur utama aplikasi ini yaitu:
1. Input nama pengguna melalui kolom EditText.
2. Tombol untuk menampilkan pesan sapaan sesuai nama yang diinput.
3. Switch untuk mengubah tema aplikasi antara mode terang dan mode gelap secara manual.

Cara menjalankan :
1. Download atau clone repository ini ke laptop.
2. Buka folder proyek menggunakan Android Studio.
3. Tunggu hingga Gradle Syns selesai.
4. Hubungkan perangkat Android (HP) atau pakai emulator.
5. Klik tombol run di bar atas Android studio.

Setelah klik tombol run, aplikasi akan melalui proses instalasi ke perangkat hp atau emulator, setelah berhasil terinstalasi masukkan nama pada kolom yang tersedia, lalu klik tombol button untuk melihat pesan sapaan muncul di layar.
geser tombol switch untuk mencoba transisi tema aplikaso.
